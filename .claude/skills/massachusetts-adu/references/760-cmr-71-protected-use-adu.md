# 760 CMR 71.00 — Protected Use Accessory Dwelling Units

**Promulgating agency:** Executive Office of Housing and Livable Communities (EOHLC)
**Authority:** St. 2024, c. 150, §§ 7-8 amending MGL Ch 40A
**Published:** Massachusetts Register, January 31, 2025
**Effective:** January 31, 2025 (regulation); February 2, 2025 (statute)
**Source:** https://www.mass.gov/info-details/accessory-dwelling-units (canonical regulatory page)

## Purpose

760 CMR 71.00 operationalizes the statutory "by-right ADU" provisions. It defines the **"Protected Use ADU"** — the subset of ADUs that receive full state preemption — and establishes the procedural and definitional framework that cities must honor.

## Key Concept — "Protected Use ADU"

Not every ADU is a "Protected Use ADU." The regulation establishes criteria that a proposed ADU must meet to qualify for by-right status under state preemption:

- **Single ADU per lot** (additional ADUs still require special permit per general zoning rules)
- **Located on a lot that permits single-family use** (typically single-family residential zones)
- **Principal dwelling exists or is being built concurrently** (cannot be a stand-alone ADU)
- **Size within statutory limits**: ≤ 900 sq ft OR ≤ 50% of primary dwelling's gross floor area, whichever is less
- **Contains all three elements of a dwelling unit**: sleeping, cooking, sanitary
- **Has a separate entrance meeting state building code egress** (direct or via shared hall/corridor)
- **Subject to local dimensional, bulk, height regulations** (these still apply, as the statute reserves them)

If a proposed ADU does NOT meet these criteria, it is NOT a Protected Use ADU and may be subject to local discretionary review (special permit, variance, etc.).

## Key Operational Provisions

### Application Process

- Local building department reviews the ADU application as it would any other by-right use — administrative, non-discretionary
- Site plan review is allowed if the city has a site plan review ordinance
- Decisions must be made within the time periods established by local building code administration (the regulation does not impose new timelines)
- Appeals follow normal zoning appeal procedures

### What Cities Must Do

- Allow Protected Use ADUs by-right in single-family residential zones
- Not impose owner-occupancy requirements
- Not impose parking requirements beyond the statutory maximum (1 space, 0 if near transit)
- Not require special permits, variances, or other discretionary approvals for a single Protected Use ADU

### What Cities May Still Do

- Require site plan review (administrative)
- Establish dimensional setbacks, bulk, height standards (reasonable)
- Establish design standards reasonably related to neighborhood character
- Regulate or prohibit short-term rental of ADUs
- Require standard building permit review, inspections, and approvals
- Collect standard permit fees
- Apply historic district rules (via historic district commission, separate authority)
- Apply wetlands and environmental protections (separate authority under MGL Ch 131)

### What Cities May NOT Do

- Require a special permit, variance, or any discretionary zoning approval for a single Protected Use ADU
- Require the owner to occupy the ADU or the principal dwelling
- Require more than 1 additional parking space (0 near transit)
- Impose a minimum lot size requirement greater than what applies to the underlying single-family use
- Impose building or occupancy requirements that effectively prohibit the use

## Non-Protected Use ADUs

An ADU that exceeds the 900 sq ft / 50% cap, or is a second/third ADU on the same lot, is NOT a Protected Use ADU. These may still be permitted under local zoning, but without the state preemption — meaning the city can impose whatever review process it wants, including requiring a special permit.

Contractor implications:
- Staying within the Protected Use envelope is worth it: by-right > special permit every time
- If the contractor wants a 1,100 sq ft ADU, they've left Protected Use — special permit applies
- If the primary dwelling is small (say 1,400 sq ft), the ADU cap is 700 sq ft (½ of primary, less than 900)

## Relationship to Other Regulations

- **780 CMR (State Building Code):** 760 CMR 71 does not override building code. An ADU must still meet structural, fire, egress, plumbing, electrical, mechanical, and energy code requirements.
- **521 CMR (Architectural Access Board):** ADU accessibility requirements depend on occupancy and whether public accommodation applies. Single-family ADUs generally are not subject to 521 CMR.
- **527 CMR (Fire Prevention):** Fire separation, smoke/CO alarms, sprinklers (if required by scope) still apply per the underlying code.
- **MA Stretch Energy Code / Specialized Opt-In Code:** New-construction ADUs must meet whichever energy code the municipality has adopted. Most MA municipalities have adopted Stretch; a growing number have adopted the Specialized Opt-In Code (requires electrification or renewable offset).
- **Local historic district rules:** Separate authority, not overridden by 760 CMR 71.
