# EOHLC Guidance Documents — Roster and Summaries

**Agency:** Executive Office of Housing and Livable Communities (EOHLC) — successor to DHCD, established 2023
**Role:** Primary state agency for housing, zoning technical assistance, and ADU guidance
**ADU portal:** https://www.mass.gov/info-details/accessory-dwelling-units

## Active Guidance Documents

Listed in approximate priority order.

### 1. Mass.gov ADU Information Page
- **URL:** https://www.mass.gov/info-details/accessory-dwelling-units
- **Purpose:** Canonical public-facing overview of the ADU law
- **Use:** Entry point for general explanations. Contains links to the FAQs, the regulation text, model bylaw, municipal checklist, and model application.

### 2. ADU FAQs
- **URL:** https://www.mass.gov/info-details/accessory-dwelling-unit-adu-faqs
- **Purpose:** EOHLC's answers to common questions from homeowners, contractors, municipalities
- **Use:** When a contractor asks a common "can I / do I have to / what if" question, check the FAQ first before composing a custom answer. The FAQ is an authoritative interpretation.

### 3. 760 CMR 71.00 — Protected Use Accessory Dwelling Units
- **URL:** https://www.mass.gov/info-details/accessory-dwelling-units (look for link to the regulation PDF)
- **Purpose:** The binding regulation itself
- **Use:** Cite for any procedural question about by-right review, Protected Use eligibility

### 4. Model ADU Zoning Bylaw
- **Partners:** EOHLC + MAPC (Metropolitan Area Planning Council) + MHP (Massachusetts Housing Partnership)
- **Purpose:** Sample bylaw text cities can adopt to come into compliance
- **Use:** When comparing a city's ADU bylaw against "best practice," the Model is the baseline. A correction invoking a non-model provision is worth scrutinizing.

### 5. Municipal Checklist
- **Partners:** EOHLC + MAPC + MHP
- **Purpose:** Compliance checklist for cities ensuring their local rules meet state minimums
- **Use:** If a city's rules don't match the checklist, flag potential preemption issues

### 6. Model ADU Application Form
- **Purpose:** Sample administrative form for ADU applications
- **Use:** Reference when a city's submittal requirements seem onerous relative to the state model

## Guidance on "Protected Use ADU" Definition

The term "Protected Use ADU" is the statutory + regulatory framing for the subset of ADUs that receive full state preemption. The key source:

- **760 CMR 71.03** (or equivalent definitions section) establishes the criteria
- See `760-cmr-71-protected-use-adu.md` for the summary

## EOHLC Office Hours

EOHLC runs periodic office hours for municipal planning boards, building departments, and practitioners. These are interpretive sessions where EOHLC answers specific questions about how the law applies. Practitioner notes from these sessions (published by MAPC, MHP, and advocacy organizations like Abundant Housing MA) are informal but informative.

- **MAPC ADU Resource Library** — https://www.mapc.org/resource-library/adu-massachusetts/
- **Abundant Housing MA** — https://www.abundanthousingma.org/
- **Massachusetts Housing Partnership (MHP)** — https://www.mhp.net/

For contractor-facing citations, prefer EOHLC and official state sources. These secondary sources are useful for interpretive context but should not appear as the primary citation in a response letter.

## How to Cite EOHLC Guidance

In a response letter:

> "EOHLC guidance clarifies [X]. See [specific guidance doc and URL]."

Do NOT cite:

> "According to Mass.gov..." (Mass.gov is a portal, not a source. Cite the specific agency and document.)

## When EOHLC Is Silent

EOHLC hasn't issued guidance on every edge case. When the regulation is silent:
- Apply the statute's plain language
- Note the interpretive gap in the response
- Recommend the contractor confirm with the local building department before committing to an interpretation
