# Massachusetts ADU Permit Application Checklist

A clean Massachusetts ADU permit submittal closes corrections-letter cycles before they start. This checklist covers what every submittal needs after Chapter 150 of the Acts of 2024 — the cover sheet declaration, the code summary, the dimensional documentation, and the energy compliance package. Use it as a final quality-check before filing.

## Cover sheet — the protected-use declaration

The cover sheet sets the regulatory frame for the entire review. The single most important addition for any post-2025 ADU permit:

> "This accessory dwelling unit is a Protected Use under MGL Ch 40A § 3 (as amended by St. 2024, c. 150, § 8) and 760 CMR 71.00."

This declaration is not legally required, but it short-circuits a class of corrections — it tells the plan checker up front that the contractor knows the law and intends to invoke state preemption on parking, owner-occupancy, and special-permit corrections. Most reviewers will adjust their approach accordingly.

[source: https://malegislature.gov/Laws/GeneralLaws/PartI/TitleVII/Chapter40A/Section3 | retrieved: 2026-05-07 | citation: MGL Ch 40A § 3 — by-right ADU paragraph after Ch 150 § 8]

[source: https://www.mass.gov/info-details/accessory-dwelling-units | retrieved: 2026-05-07 | citation: 760 CMR 71.00 — EOHLC implementing regulation]

## Code summary table

Beneath the protected-use declaration, every cover sheet should carry a code summary table listing the controlling codes and editions. The standard MA ADU table:

| Discipline | Code | Edition / Section |
|------------|------|-------------------|
| Building | 780 CMR 10th Edition | IRC 2021 base with MA amendments |
| Plumbing | 248 CMR | Current edition |
| Electrical | 527 CMR 12 | NEC 2023 with MA amendments |
| Energy | 225 CMR 22 | Base / Stretch / Specialized as adopted by municipality |
| Fire | 527 CMR 1 | Comprehensive Fire Safety Code |
| Accessibility | 521 CMR | Massachusetts Architectural Access Board |
| Smoke / CO | MGL Ch 148 §§ 26F, 26F½ | Hardwired with battery backup, new construction |

[source: https://www.mass.gov/state-building-code | retrieved: 2026-05-07 | citation: 780 CMR — MA State Building Code, 10th Edition]

The table tells the plan checker the contractor knows the code stack. Missing or wrong-edition references invite corrections. "780 CMR 9th Edition" on the cover is an immediate red flag — the 10th Edition has been current since October 2023.

## Project identification

Standard project-identification block on the cover sheet:

- Project name: "Accessory Dwelling Unit — [address]"
- Property address: full street, city, state, zip
- Assessor's parcel ID (map / block / lot)
- Zoning district (verify against city GIS — do not project from neighboring parcels)
- Date of drawings, revision number, designer's stamp and signature

The assessor's parcel ID is required by most MA cities for cross-referencing in their permit-tracking systems. Address alone is not enough.

## Project type / scope statement

Three lines that resolve a class of common corrections:

1. **ADU classification:** new-construction detached / new-construction attached / interior conversion / exterior conversion. Be specific.
2. **Reference to by-right protected-use status** under Ch 150 of 2024. Recommended, not required.
3. **Scope summary:** "Construct a [N] sqft [type] accessory dwelling unit with [electrical / plumbing / mechanical scope]."

Common correction without this section: "Project scope is unclear; please clarify whether this is new construction or alteration." The fix is naming the work in plain terms on the cover sheet.

## Design professional identification

For projects requiring a registered architect or licensed engineer (most new-construction ADUs in MA do), the cover sheet must include:

- Design professional's full name
- License type and Massachusetts license number
- Professional stamp / seal in ink (paper) or scanned (digital)
- Signature and date

[source: https://malegislature.gov/Laws/GeneralLaws/PartI/TitleXVI/Chapter112 | retrieved: 2026-05-07 | citation: MGL Ch 112 §§ 60A (architects) and 81D (engineers) — licensure and seal]

The Massachusetts homeowner exemption under MGL Ch 112 is narrower than many homeowners assume. For an ADU project, plan to stamp it with a registered design professional unless the work is purely administrative (no structural changes, no new construction). When in doubt, stamp it.

## Dimensional documentation

Every Massachusetts ADU plan needs the following dimensional documentation, typically on the site plan:

- **Lot area** with the assessor's record value and the plan's measured value (they should match within 1%)
- **Existing primary dwelling footprint** dimensioned to all property lines
- **Proposed ADU footprint** dimensioned to all property lines
- **Combined lot coverage** with the city's district maximum noted
- **Combined FAR** with the district maximum noted (cities that use FAR — Cambridge, Somerville, Brookline, parts of Boston)
- **Setback dimensions** on all four sides for both structures
- **Fire-separation distance** from ADU exterior walls to nearest lot line(s)

For attached ADUs, also show the rated wall or floor/ceiling assembly with UL listing reference per 780 CMR R302.3.

## Energy compliance documentation

The single most-skipped correction-bait on Massachusetts ADU permits. Every submittal needs:

- **Identified compliance path** — base 225 CMR 22, Stretch Code, or Specialized Opt-In Code based on what the municipality has adopted
- **HERS rater** identified by name and certification, with planned target rating
- **Envelope summary** — wall R-values, ceiling R-values, foundation R-values, window U-factors and SHGC
- **Mechanical system summary** — heat pump make/model proposed (or equivalent for Specialized Code municipalities), water heating method, ventilation method per IRC M1505

[source: https://www.mass.gov/info-details/specialized-stretch-code | retrieved: 2026-05-07 | citation: 225 CMR 22 — MA Energy Code (Base / Stretch / Specialized Opt-In tiers)]

Cities on the Specialized Opt-In Code include Boston, Cambridge, Somerville, Newton, Brookline, and approximately 60 other MA municipalities. Verify your city's adopted tier on the MA Department of Energy Resources published list.

## Smoke and CO alarm placement

Required on every new ADU permit:

- Smoke alarms in every sleeping room
- Smoke alarms outside each sleeping area
- Smoke alarms on every story including basements
- CO alarms within 10 feet of each sleeping area on every story (if fuel-burning appliances or attached garage)
- All hardwired with battery backup for new construction
- Interconnection per IRC R314

[source: https://malegislature.gov/Laws/GeneralLaws/PartI/TitleXX/Chapter148 | retrieved: 2026-05-07 | citation: MGL Ch 148 §§ 26F (smoke) and 26F½ (CO) — alarm requirements]

For all-electric ADUs (Specialized Opt-In Code municipalities) without an attached garage, CO alarms are not required by the statute but are recommended.

## Special-condition declarations

Three checkboxes that resolve a class of preemption corrections up front:

- **MBTA / qualifying-transit proximity** — if the parcel is within 0.5 miles walking distance of a commuter rail station, subway stop, ferry terminal, or designated bus station, declare it on the cover sheet. Triggers the 760 CMR 71.00 parking exemption.
- **Historic district status** — if the parcel is in a local historic district under MGL Ch 40C, declare it. Triggers Historic Commission review on the parallel track.
- **Wetlands / floodplain status** — if within 100 feet of a regulated wetland or in a FEMA flood zone, declare it. Triggers Conservation Commission and FEMA-zone construction requirements.

## Pre-submittal final check

Before filing, run through this list:

- [ ] Cover sheet protected-use declaration present
- [ ] Code summary table with current editions
- [ ] Project identification block complete
- [ ] Scope statement names the ADU classification
- [ ] Design professional stamp and signature present
- [ ] Lot coverage and FAR calculations shown vs district maximums
- [ ] All setbacks dimensioned
- [ ] Fire-separation distance on site plan
- [ ] Rated wall/floor assembly detail for attached ADUs
- [ ] Energy compliance path identified, HERS rater named
- [ ] Smoke and CO alarm placement on plans
- [ ] MBTA proximity declared if within 0.5 mi
- [ ] Historic district status declared
- [ ] Wetlands / floodplain status declared
