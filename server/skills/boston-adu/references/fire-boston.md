# Boston Fire Department Review for ADUs

## What BFD Reviews

Boston Fire Department (BFD) is a direct city service — not a regional or county fire district. BFD performs fire-life-safety review on residential building permits in addition to ISD's building-code review. For an ADU project, BFD review covers:

- Sprinkler triggers and design (where required by 780 CMR Chapter 9 or local policy)
- Fire department access (apparatus reach to the structure)
- Fire flow and water supply (hydrant proximity, residual pressure)
- Fire-resistance-rated separations between the ADU and primary dwelling
- Smoke and CO alarm compliance under MGL Ch 148 §§26F and 26F½ (BFD inspects to certificate-of-compliance for sale/transfer transactions, not building permits)
- Means of egress for emergency evacuation
- Fire alarm system design (where required)

## Sprinkler Triggers — State Floor

Massachusetts is one of the few states where IRC R313 (residential sprinklers) was NOT adopted statewide. 780 CMR 10th Edition (effective Oct 2023) does not require sprinklers in one- and two-family dwellings as a baseline. Sprinklers are triggered by specific conditions:

- **Existing sprinkler in the primary dwelling**: if the primary is sprinklered, additions and ADUs typically extend the sprinkler system
- **Townhouse / multi-family classification**: if the ADU configuration triggers IBC Group R-2 multi-family classification (3+ units typically), NFPA 13R sprinklers apply
- **Local ordinance**: cities may adopt sprinkler requirements above the state floor for specific occupancies or districts
- **Substantial alteration thresholds**: gut renovation of an existing structure may trigger sprinklers depending on scope
- **Unusual configurations**: some BFD plan reviewers require sprinklers in ADUs in dense urban contexts even where 780 CMR does not. This is a discretionary judgment with limited statutory backing; verify per project.

## When a Boston ADU Project Triggers Sprinklers

Default analysis for the agent:

| Configuration | Sprinkler default | Verify |
|---|---|---|
| Single detached ADU under 900 sq ft, primary unsprinklered, lot in standard residential district | No sprinkler required | BFD plan reviewer comments |
| Single detached ADU, primary sprinklered | Sprinkler required (extend existing system) | Verify connection point and design via licensed sprinkler designer |
| Attached ADU as addition to unsprinklered primary | No sprinkler required for addition; verify fire separation per IRC R302.3 | BFD plan reviewer comments |
| Attached ADU as addition to sprinklered primary | Extend sprinkler into addition | Sprinkler designer |
| Conversion of attic / basement / garage to ADU | Generally no sprinkler unless primary is sprinklered or local ordinance triggers | BFD plan reviewer comments |
| 3-or-more-unit configuration (primary + 2 ADUs, or 2-family + ADU) | NFPA 13R typically required (IBC R-2) | BFD + sprinkler designer |
| Any project in a high-density urban context | Default to BFD discretionary review even where state floor doesn't require sprinklers | BFD pre-application meeting |

When a Boston corrections letter demands a sprinkler installation, the agent's response path is:

1. Identify the project configuration from the plans and corrections.
2. Match against the table above.
3. If the corrections demand exceeds the state floor and exceeds the configuration's typical trigger, request the BFD plan reviewer cite the specific 780 CMR / 527 CMR / Boston ordinance section justifying the requirement.
4. If the demand is within the state floor (e.g., primary is sprinklered), confirm and budget accordingly.

Do NOT flat-assert "no sprinkler required" without checking the configuration. The cost of a sprinkler installation in a Boston ADU is substantial; getting this wrong is a costly contractor error.

## Fire Department Access

Boston is dense; BFD apparatus access is a frequent corrections topic. Access requirements derive from 780 CMR Chapter 5 (means of egress) and BFD policy:

- **Fire department apparatus access road**: required to within 50–150 ft of the primary structure, depending on building height and configuration. Single-family residential typically OK with public-street access.
- **Access width**: BFD apparatus requires a clear width (typically 20 ft for two-way, narrower for one-way) and overhead clearance (typically 13.5 ft).
- **Turning radius**: apparatus must be able to maneuver — not block on dead-end driveways without a turnaround.

For most Boston ADU projects on existing single-family lots with public-street primary access, fire department access is not a corrections issue. **Detached ADUs on rear-of-lot lots accessed by easement, narrow driveway, or no driveway at all** are where access corrections appear most often.

## Fire Flow and Water Supply

Fire flow corrections involve hydrant proximity, residual pressure, and available volume. BFD coordinates with Boston Water and Sewer Commission (BWSC) on hydrant placement and water-supply adequacy.

For most Boston ADU projects (which sit in dense neighborhoods with extensive hydrant networks), fire flow is adequate by default. **Outer neighborhoods (Hyde Park, West Roxbury, Roslindale, Mattapan) have sparser hydrant placement** and may require fire-flow analysis for new structures, especially detached ADUs more than ~500 ft from the nearest hydrant.

When a corrections letter demands a fire-flow study, the test is:

1. Distance from proposed ADU to nearest hydrant
2. Residual pressure (typically required minimum 20 psi at 1,500 gpm)
3. Static and residual pressure available at the hydrant under flow

If the project sits in a high-density inner-city block, the requirement is usually a routine demonstration via the fire flow data already on file with BWSC. If outer-neighborhood, an actual hydrant flow test may be needed.

## Smoke and CO Alarms — The MGL Ch 148 §§26F / 26F½ Layer

Massachusetts has state-level smoke detector and CO alarm requirements under MGL Ch 148 §§26F and 26F½ that apply at TWO points:

1. **At construction**: 780 CMR / IRC R314, R315 dictate the design (interconnected, hardwired, battery backup, locations).
2. **At sale or transfer**: MGL Ch 148 §§26F / 26F½ require BFD inspection and a Certificate of Compliance before the dwelling can be sold or transferred. ADUs are subject to this on a per-dwelling-unit basis.

For corrections letters at the building permit stage, the relevant code is 780 CMR / IRC R314 / R315. The Ch 148 inspection layer comes at sale/transfer, not at construction.

## Workflow for the Agent

When a Boston ADU correction touches fire-life-safety:

1. **Identify the BFD corrections topic**: sprinkler / access / fire flow / fire separation / alarms / fire alarm system.
2. **For sprinkler**: classify the project per the table above, match to expected trigger, push back if BFD demand exceeds.
3. **For access**: verify primary access path; if rear-of-lot or easement-served, expect corrections.
4. **For fire flow**: check hydrant proximity in dense vs. outer neighborhood; plan for fire-flow test in outer neighborhoods.
5. **For fire separation (attached ADUs)**: cite IRC R302.3 (1-hour fire-rated assembly between primary and ADU); verify wall, floor/ceiling continuity in plans.
6. **For alarm requirements**: distinguish construction-stage IRC R314/R315 from sale-stage MGL Ch 148 §§26F/26F½; the corrections letter is almost always about construction-stage compliance.
7. **Cite both 780 CMR / IRC sections AND the Boston enforcement context inline.**
