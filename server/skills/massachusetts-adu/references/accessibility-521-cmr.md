# 521 CMR — MA Architectural Access Board (Accessibility for ADUs)

**Code:** 521 CMR — Rules and Regulations of the Massachusetts Architectural Access Board
**Authority:** Massachusetts Architectural Access Board (MAAB)
**Source:** https://www.mass.gov/orgs/architectural-access-board [source: https://www.mass.gov/orgs/architectural-access-board | retrieved: 2026-05-04 | citation: MAAB authority over 521 CMR]

## Scope For ADU Analysis

521 CMR governs architectural accessibility in Massachusetts. The headline question for an ADU permit: **does 521 CMR require the ADU to be accessible?**

**Default answer: usually no, for ADUs accessory to single-family dwellings.** But several exceptions can flip the answer. The agent must check, not assume.

## When 521 CMR Triggers For An ADU

521 CMR uses scope categories. The relevant ones for ADUs:

### Category 1 — Single-Family Dwellings

521 CMR has historically exempted single-family residences from full accessibility requirements. An ADU accessory to a single-family primary dwelling, where neither unit is publicly accessible, **typically does not trigger 521 CMR scoping**. [source: https://www.mass.gov/regulations/521-CMR-architectural-access-board | retrieved: 2026-05-04 | citation: 521 CMR scoping for one- and two-family dwellings]

### Category 2 — When 521 CMR Can Trigger

Even on a single-family parcel, accessibility may apply if:

- **The ADU is rented** as a separate dwelling unit AND the project crosses certain thresholds. The MAAB's variance procedure addresses rental units; check the current scoping decision tree. Practitioners frequently obtain MAAB variances for ADU rental scenarios.
- **The project includes site work that creates a public-accessible path** (e.g., a shared entry walk that serves both the primary and ADU, or that the public accesses).
- **The construction cost crosses a threshold** that triggers full code compliance retroactively. The threshold has historically been 30% of full and fair cash value for substantial alterations. Confirm the current threshold against the regulation. [source: https://www.mass.gov/doc/521-cmr-1-introduction/download | retrieved: 2026-05-04 | citation: 521 CMR Section 3.3 — work-cost-threshold scoping]

### Category 3 — Variance Procedure

If 521 CMR scoping suggests accessibility applies but the ADU layout cannot reasonably accommodate it (e.g., conversion ADU in a basement with limited headroom and structural constraints), the project applies for a MAAB variance.

**The variance must be filed with the local building department permit application.** Variances are heard by MAAB at scheduled meetings; expect 4-12 weeks. Plan ahead for the schedule.

## Practical ADU Implications

### Most Single-Family ADUs

For most ADUs accessory to a single-family primary, **521 CMR is silent** and the ADU does not need to be accessible. The agent should still note this in the eligibility analysis to avoid contractor confusion.

### Conversion ADUs

Conversion ADUs (basement, attic, garage) often raise the work-cost threshold question if the project total exceeds 30% of property fair cash value. Get the assessor's value early.

### Rental ADUs

Rental of an ADU does not automatically trigger 521 CMR for residential properties with fewer than 4 units. Confirm against current MAAB guidance — the regulation interacts with both 521 CMR and federal Fair Housing Act.

## Common Corrections

- City building department flags accessibility for an ADU that's clearly out of 521 CMR scope (single-family primary, no public access). Response: cite the scope rule and request release of the correction.
- Conversion ADU project cost approaches 30% of FCV without a variance application. Response: file MAAB variance promptly to avoid permit hold.
- ADU shared site work creates an accessible-path issue. Response: detail the path on the site plan with the standard MAAB-compliant section.

## When To Engage A Specialist

- If the ADU is part of a multi-family project (3+ dwelling units total on the parcel), full 521 CMR scoping applies — not an ADU question, get a code consultant.
- If the local building department is asserting accessibility scope and the contractor disagrees, request the MAAB scoping memo or apply for a variance proactively.
