# 225 CMR 22 — MA Energy Code (Base + Stretch + Specialized Opt-In)

**Code:** 225 CMR 22 — Massachusetts Energy Code
**Editions in force:** Base Code, Stretch Code, Specialized Opt-In Code (the three-tier structure)
**Authority:** Department of Energy Resources (DOER), Board of Building Regulations and Standards (BBRS)
**Source:** https://www.mass.gov/info-details/stretch-energy-code-development-2022 [source: https://www.mass.gov/info-details/stretch-energy-code-development-2022 | retrieved: 2026-05-04 | citation: DOER Specialized Opt-In Code portal]

## Scope For ADU Analysis

The energy code that applies to a given ADU permit depends on which version the municipality has adopted. All three are reachable for the same parcel depending on the date and city. Get the city's adoption status WRONG and the entire envelope and HVAC scope is wrong.

Three tiers, in increasing stringency:

1. **Base Code** — IECC 2021 with MA amendments. Applies in cities that have not adopted Stretch.
2. **Stretch Code** — Statewide adopted by ~300 communities. Tightens envelope, raises HERS rating, adds renewable provisions.
3. **Specialized Opt-In Code** (225 CMR 22 Appendix RC) — Adopted by Boston, Cambridge, Somerville, Newton, Brookline, and ~30 other municipalities. Tightest tier; effectively all-electric or near-zero-energy.

[source: https://www.mass.gov/info-details/stretch-energy-code-development-2022 | retrieved: 2026-05-04 | citation: DOER three-tier structure]

## How To Determine Which Code Applies

Single source of truth: DOER publishes a list of adopting municipalities by code version. Always check the date — adoptions occur on a rolling basis. [source: https://www.mass.gov/info-details/massachusetts-municipalities-that-have-adopted-the-stretch-code | retrieved: 2026-05-04 | citation: DOER Stretch Code adoption list]

If the city has adopted the Specialized Opt-In Code, treat it as the operative code for ADU envelope and systems compliance. If only Stretch, that's the operative code. Default to Base if no adoption listed.

## Specialized Opt-In Code (225 CMR 22 Appendix RC) — Practical ADU Implications

Three compliance pathways for new low-rise residential (which includes ADUs):

### Pathway 1 — All-Electric

- All space heating, water heating, cooking, and clothes drying via electricity (typically heat pumps for HVAC and water heating).
- No fossil-fuel combustion appliances on the lot.
- Envelope per Stretch Code R-values (still required as a baseline).
- HERS rating ≤ 45 (verifier-confirmed) for new construction; ≤ 55 for substantial renovation.

### Pathway 2 — Mixed-Fuel + Solar PV Offset

- Allowed to retain natural gas for cooking, water heating, or one space-heating zone, BUT
- Must install rooftop solar PV sized to offset the additional fossil-fuel energy use.
- Solar-ready provisions required (conduit pathway, structural roof load capacity, electrical panel headroom). [source: https://www.mass.gov/info-details/specialized-stretch-code | retrieved: 2026-05-04 | citation: 225 CMR 22 Appendix RC solar offset pathway]
- HERS rating ≤ 42 (tighter, to compensate for fossil fuel use).

### Pathway 3 — Passive House

- Certified Passive House (PHIUS or PHI) construction.
- Envelope substantially exceeds Stretch.
- Air leakage ≤ 0.6 ACH50 tested.
- Allowed regardless of fuel source — Passive House envelope offsets the fuel choice.

## Envelope R-Value Targets (Specialized Opt-In, Climate Zone 5 — most of MA)

For ADU plan-review purposes, the agent should expect to see these called out on the building section:

- Wall: R-21 cavity + R-5 continuous (≈R-26 effective)
- Roof / ceiling: R-49 to R-60
- Floor / slab edge: R-19 cavity + R-10 perimeter
- Window U-factor: ≤ 0.27
- Air leakage target: ≤ 3.0 ACH50 tested

[source: https://www.mass.gov/info-details/specialized-stretch-code | retrieved: 2026-05-04 | citation: 225 CMR 22 Appendix RC Climate Zone 5 envelope]

## Stretch Code (Without Specialized) — Practical ADU Implications

If the city is on Stretch but has NOT adopted Specialized Opt-In:

- Envelope tightened relative to Base IECC 2021 but no all-electric mandate.
- HERS rating typically ≤ 55 for new construction.
- Solar-ready provisions required as of 2023 amendment.

## Base Code — Practical ADU Implications

The IECC 2021 baseline (with MA amendments). Applicable when city has NOT adopted Stretch.

- Envelope per IECC 2021 Climate Zone 5 (typical MA).
- Wall R-13 cavity + R-5 continuous, OR R-20 cavity (alternative path).
- Window U-factor ≤ 0.30.
- Air leakage ≤ 5.0 ACH50.

## Common Corrections

- HERS rater not engaged (Stretch and Specialized both require HERS verification on new construction)
- Wall R-value cited at R-19 (Base Code) when Specialized requires R-21 + R-5
- Mixed-fuel ADU proposed in a Specialized-Opt-In municipality without solar PV offset analysis
- Specialized-Opt-In compliance form (RES-Check or HERS report) not submitted with permit
